代码功能

一个python的文件有两种使用方法，1、可以被直接运行，2、作为模块（也就是库），被其他.py文件导入。if name == ‘main’:的作用就是控制这两种方法代码的执行情况。if name == ‘main’:下的代码只有在第一种方法（即作为脚本直接运行）的时候，才会执行，import到其他脚本（.py)中时不会执行的。

运行原理

每个python文件都包含内置变量name,当文件被直接运行的时候name为main,也相当于当前的文件名（包含后缀.py）;如果被作为模块import到其他文件（.py）中，则name为模块名称，不满足if name == ‘main’:这个条件，所以无法执行后面的代码。
